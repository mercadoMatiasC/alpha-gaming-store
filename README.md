
# AlphaGaming Store

AlphaGaming Store es un e-commerce enfocado en productos gaming, desarrollado como proyecto de aprendizaje.  
Estado: Alpha (en desarrollo activo).  
Stack principal: Laravel.  
Objetivo: aplicar buenas prácticas de desarrollo web, arquitectura limpia y control de versiones.
